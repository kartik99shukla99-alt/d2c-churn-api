# Monitoring Plan

## Model Monitoring

- Track daily prediction counts
- Track churn prediction percentage
- Monitor API latency
- Monitor prediction failures

## Data Quality Checks

- Missing values
- Invalid feature ranges
- Unexpected categorical values

## Responsible AI

- Do not use sensitive personal data
- Human review for high-risk decisions
- Explain prediction limitations
- Regular retraining and validation

## Alerts

- API downtime
- High error rate
- Sudden prediction drift
